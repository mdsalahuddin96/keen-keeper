# 🌟 Keen Keeper   
Keen Keeper is a smart friend relationship tracking web application that helps you maintain meaningful connections by reminding you when to reach out. It allows you to monitor your interactions, stay consistent, and never lose touch with important people.

# ⚙️ Technologies  
- Next.js (App Router)

- React

- Tailwind CSS

- Vercel (Deployment)

# 🚀 Key Features  
**📊** **Smart Contact Status Tracking**

Track your friends using Overdue, Almost Due, and On-Track statuses, ensuring you stay connected at the right time.

**📞** **Interactive Communication & Timeline**

View friend details and interact via Call, Text, or Video. All interactions are stored and displayed in a timeline view.

**📈** **Insights & Activity Analytics**

Visualize your communication patterns with charts showing call, text, and video activity, with filtering options for better insights.
